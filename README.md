# 🧠 心理学刷题宝典

一个功能完善的心理学刷题网页应用，包含1526道题目。

## 📊 题库统计

| 题型 | 数量 |
|------|------|
| 单选题 | 961道 |
| 多选题 | 169道 |
| 判断题 | 396道 |
| **合计** | **1526道** |

## ✨ 功能特点

- ✅ 顺序刷题/随机刷题
- ✅ 错题重做功能（答对后自动移除）
- ✅ 收藏题目
- ✅ 模拟考试（50题/次）
- ✅ 学习统计
- ✅ 用户登录系统
- ✅ 数据备份/恢复
- ✅ 键盘快捷键支持

## 🚀 快速开始

### 本地使用

1. 双击 `start-server.bat` 启动本地服务器
2. 在浏览器中访问 http://localhost:8080
3. 注册账号并登录
4. 开始刷题！

### 公网部署

详见 `部署指南.txt` 或 `一键部署.bat`

## ⌨️ 键盘快捷键

- `A/B/C/D` 或 `1/2/3/4` - 选择选项
- `E/F` - 选择E/F选项（多选题）
- `Enter` - 提交答案/下一题
- `←/→` - 上一题/下一题

## 📊 数据隔离

✅ **错题库已经按用户分开！**

每个用户的数据是独立存储的：
- 用户A的错题只在用户A的账号下
- 用户B的错题只在用户B的账号下
- 不会出现错题混淆的情况

数据存储在浏览器的 localStorage 中，按用户名区分。
